Privacy & Compliance

بيانات عملاءك في أمان كامل

احترامنا لخصوصية عملاءك مبني في الأداة من اليوم الأول — مش feature إضافي. كل بيانات الـ PII بتتم معالجتها بمعايير GDPR و iOS 14+ requirements.

🔐 SHA-256 HASHING
Hashing تلقائي قبل أي إرسال

الأداة بتعمل SHA-256 hashing لكل بيانات العميل (Email · Phone · First Name · Last Name) من جوّا المتصفح قبل ما تتبعت لأي منصة. النصوص الـ raw مش بتغادر جهاز العميل أبداً.

  • ✓بيانات raw لا تُرسل أبداً — كل شيء هاش client-side عبر crypto.subtle.
  • ✓تطبيع البيانات قبل الـ hash (lowercase + trim + E.164 للأرقام).
  • ✓متوافق مع متطلبات Meta CAPI · TikTok Events API · Snapchat CAPI · GA4 Enhanced Conversions.
  • ✓متوافق مع GDPR · CCPA · ISO 27001 وسياسات حماية البيانات الإقليمية.
// تلقائياً لكل event: mixpanel.people.set({ $email: 'a1b2c3...', // SHA-256 $phone: '9d8e7f...', // SHA-256 external_id: 'ext_abc123' });
📡 GTM SERVER-SIDE
Server-Side Tracking Compatible

الـ Container المولّد جاهز للـ GTM Server-Side من اليوم صفر — كل event بيتبعت بـ event_id فريد ومعرّف external_id ثابت لربط الـ Browser Hits بالـ Server Hits بدقة 100%.

  • ✓Deduplication كامل: Browser → Server بنفس الـ event_id — منعاً للـ double counting.
  • ✓external_id منفصل تماماً عن user_id — مفيش خلط بين معرف المنصة ومعرف التطبيق.
  • ✓الـ Cookie external_id بيتقرأ تلقائياً في كل event للـ cross-domain tracking.
  • ✓دعم تلقائي للـ Click IDs (fbclid · gclid · ttclid · scclid) لـ attribution كاملة.
  • ✓متوافق مع Meta CAPI · TikTok Events API · Snapchat CAPI · GA4 Measurement Protocol · Mixpanel Server-Side.
// كل tag بيشمل تلقائياً: fbq('track', 'Purchase', props, { eventID: '{{DL - event_id}}', userData: { external_id: '{{Cookie - external_id}}', em: '{{customer.emailHashed}}' } });
